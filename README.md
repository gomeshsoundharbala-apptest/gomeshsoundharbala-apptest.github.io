# gomesh.

Personal one-pager — dark, contemporary, zero-build. The hero hides a small
game: a shuttlecock falls with shuttle-style air drag and your cursor is the
racquet. Swing at it to keep a rally going.

## Stack

One `index.html`. No frameworks, no build step. Canvas physics, Space Grotesk
+ Inter, scroll-reveal via IntersectionObserver, `prefers-reduced-motion`
respected.

## Hosting (GitHub Pages)

Already wired: push to `main` and GitHub Pages serves the root of the repo.

To run locally, just open `index.html` in a browser.
